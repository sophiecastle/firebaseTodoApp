# Firebase To-Do List
A simple To-Do List web application built using JavaScript and Firebase.

## Features
- Add tasks
- View tasks in real-time
- Delete tasks
- Mark tasks as completed
- Data stored in Firebase Firestore
- Real-time sync using Firebase Realtime Database

## Technologies Used
- HTML
- Bootstrap 5
- JavaScript
- Firebase Firestore
- Firebase Realtime Database

## Notes
- Firestore is used as the primary data store for todos
- Realtime Database is used to notify clients of updates
- The UI is built using Bootstrap for simplicity and clarity

## How to Run
1. Download or clone the repository
2. Open the HTML file in a browser
